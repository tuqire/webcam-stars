# WEBGL Webcam Stars

This project allows the user to type text and see the stars on the screen formulate them. Built using `THREE.js`, and `GPGPU`.

## Run locally

```bash
yarn start
```

Then visit [http://localhost:8089](http://localhost:8089)

## Working Demo
